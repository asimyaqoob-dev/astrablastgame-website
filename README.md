## Upload Instructions:

1. Go to your GitHub repository: https://github.com/asimyaqoob-dev/astrablastgame-website

2. Delete the old HTML files if they exist

3. Click "Add file" → "Upload files"

4. Drag and drop ALL the files listed above

5. Scroll down and click "Commit changes"

6. Wait 2-3 minutes for GitHub Pages to rebuild

7. Visit your site to see the changes!
